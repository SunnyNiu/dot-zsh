# dot-zsh
Personal Zsh config

## Usage
### Clone to ~/.zsh folder
```git clone git@github.com:ouchxp/dot-zsh.git ~/.zsh```

### Install plugins and create links for .zshrc and .zshenv
```sh ~/.zsh/install.sh```

## Optional Scripts
### Install brew and packages
```sh ~/.zsh/brew.sh```

### Create SSH key
```sh ~/.zsh/ssh.sh```
